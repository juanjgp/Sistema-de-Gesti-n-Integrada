Sistema de gestión integrada

Antecedentes

En varias empresas cada uno de los departamentos, tiene un sistema para almacenar y tratar los datos. Por ejemplo, un departamento utiliza hojas de Google Sheets, otro utiliza Excel, otro un programa distinto de terceros, etc. El sistema de gestión integrada SGI pretende unificar en una empresa todos estos archivos en un entorno único para el trabajo conjunto. En el video que se encuentra en esta sección se explica el ejemplo de una empresa que utiliza archivos separados y lo que se propone.

Funcionamiento ejemplo empresa

Departamento producción

Anota el proceso del crecimiento del champiñón

Usa una hoja Excel para anotar condiciones de crecimiento Tª, Humedad...

Departamento recolección

Recolecta y anota los envases de productos

Usa una hoja Excel para anotar los envases y las cantidades recolectadas. Distinta hoja que producción.

Problema: si hay producto defectuoso, para consultar las condiciones debe usar la otra hoja Excel de producción

Departamento almacén

Anota el stockaje de envases obtenidos en la recolección

Usa una hoja de Google Sheet para anotar stock

Problema: debe consultar la hoja Excel de recolección para comprobar y regular el stock

Departamento comercial

Anota los pedidos de los clientes y los envia a almacén para prepararlos

Usa una hoja de Google Sheet para anotar los pedidos pendientes y completados

Problema: debe consultar la hoja de stock en otro fichero para comprobar y regular el pedido con el stock

Distribución de datos

Como se muestra en la imagen a modo de esquema. Todas las secciones acceden a la misma base de datos o entorno de trabajo y trabajan todos sobre la misma.

Esta se encuentra alojada en un servidor central, de manera que los datos de cada sección se encuentran actualizados y listos para su tratamiento.
